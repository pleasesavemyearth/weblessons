### Ⅰ.실습목표

1. PHP와 MySQL을 연동하여 웹 기반의 애플리케이션을 개발할 수 있다.
1. PHP의 기본문법과 SQL 기본질의어를 활용할 수 있다.

### Ⅱ.구현 코드

1. 데이터베이스 생성 
1. 애플리케이션용 계정 생성 및 권한부여
1. 테이블 생성
1. 데이터 추가
1. 데이터 조회
1. 데이터 갱신
1. 데이터 삭제

### Ⅲ. 제약사항

1. PHP와 MySQL 연동은 MySQLi Object Oriented
